Creates a panel that holds controllers.
Parameters:
Name Type Attributes Description
options object <optional>
Properties
Name Type Attributes Default Description
autoPlace boolean <optional>
 true Adds the GUI to `document.body` and fixes it to the top right of the page.
container HTMLElement <optional>
 Adds the GUI to this DOM element. Overrides `autoPlace`.
width number <optional>
 245 Width of the GUI in pixels, usually set when name labels become too long. Note that you can make name labels wider in CSS with `.lil‑gui { ‑‑name‑width: 55% }`
title string <optional>
 Controls Name to display in the title bar.
injectStyles boolean <optional>
 true Injects the default stylesheet into the page if this is the first GUI. Pass `false` to use your own stylesheet.
touchStyles number <optional>
 true Makes controllers larger on touch devices. Pass `false` to disable touch styles.
parent GUI <optional>
 Adds this GUI as a child in another GUI. Usually this is done for you by `addFolder()`.
